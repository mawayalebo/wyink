# Wyink

This is a Wikipedia Search React web app.

Preview link
-------
wyink.netlify.app


Clone the repo to get started:

```properties
git clone https://github.com/mawayalebo/wyink
```

> `cd` into folder

Install npm dependencies:
```
npm install
```

Finally, run the web app: 
```
npm run start
```
